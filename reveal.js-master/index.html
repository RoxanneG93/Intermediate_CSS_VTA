Pixels VS Ems

  • When defining the font-size property, an em is equal to the size of the font that applies to the paretn of the element in question.
  • Unless specified anywhere else on the page, the browser's font-size default is typically 16px.
  • By default: 1em = 16px, and 2em = 32px

Check out MDN's explanation font-size if further explanation needed.
